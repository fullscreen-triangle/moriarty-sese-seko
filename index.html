What is Moriarty?

Moriarty is a comprehensive framework for analyzing sports videos through advanced computer vision techniques to extract detailed biomechanical data. Designed for coaches, athletes, and researchers, it provides unprecedented insights into athletic performance and technique optimization.

Installation & Setup

Prerequisites

  • Python 3.8+ with pip
  • CUDA-capable GPU (recommended for optimal performance)
  • FFmpeg for video processing
  • 50GB+ disk space for video storage and results

Quick Installation

# Clone the repository git clone https://github.com/yourusername/moriarty-sese-seko.git cd moriarty-sese-seko # Create and activate virtual environment python -m venv .venv source .venv/bin/activate # On Windows: .venv\Scripts\activate # Install dependencies pip install -r requirements.txt # Run setup script bash setup.sh

Command Line Usage

# Analyze a single video python -m src.cli.main analyze --input videos/sprint.mp4 --output results # Batch process multiple videos python -m src.cli.main batch --input-dir videos --output-dir results --pattern "*.mp4" # Start distributed orchestration system python -m orchestrator.main --workers 4 --memory-limit 0.6
